1. index.html:
        This is the main HTML file for a simple color flipper application.
        It includes a button to change the background color to a random simple color.
        It uses the app.js JavaScript file and style.css for styling.

2. hex.html:
        This HTML file is for a hex color flipper variant of the same application.
        It includes an input field where users can enter a hex color code.
        It uses hex.js JavaScript file for functionality and the same style.css for styling.

3. style.css:
        The CSS file linked in your HTML files for styling. This file is shared across the different HTML files.

4. app.js and hex.js:
        These JavaScript files are used in index.html and hex.html respectively to handle the functionalities such as changing colors and handling user inputs.
